# Danganronpa S: Ultimate Summer Camp - CE Table

## 简介
本项目为《幸福枪弹辩驳S》（Danganronpa S: Ultimate Summer Camp）的 Cheat Engine 修改表（.CT 文件）。本项目作为个人学习 Cheat Engine 逆向与内存修改的实践产物，旨在为单机模式提供数值辅助功能，以减少重复游玩的时间成本。

## 功能列表 (Features)
目前已实装以下功能：

- **全局资源**：
  - 无限抽卡资源

- **培养模式专属**：
  - 无限技能碎片
  - 无限商店金币
  - 锁定培养天数（停止回合流逝）*
  - 立即结束培养（快速跳过流程进行结算）*

> **\* 重要使用前置条件**：使用“锁定培养天数”与“立即结束培养”功能时，**必须在进入培养模式后，先让游戏内的时间自然流动一天，然后再勾选激活这两个选项**。如果在初始第一天直接激活功能，会导致游戏逻辑冲突并造成程序卡死。

## 环境要求 (Requirements)
- **操作系统**：Windows 环境
- **依赖软件**：[Cheat Engine](https://www.cheatengine.org/)
- **适用游戏**：《幸福枪弹辩驳S》 Steam 最新版本

## 使用方法 (Usage)
1. 启动《幸福枪弹辩驳S》游戏进程。
2. 运行 Cheat Engine 软件。
3. 点击 Cheat Engine 左上角的计算机图标（Select a process to open），选择游戏进程（通常为 `Danganronpa S.exe`）。
4. 将下载的 `.CT` 文件拖拽至 Cheat Engine 窗口，或通过菜单栏 `File -> Load` 导入文件。
5. 在下方地址列表中，勾选对应功能左侧的激活框（Active）使修改生效（请务必遵循上述的“重要使用前置条件”）。

## 注意事项 (Disclaimer)
- 本修改表仅供单机学习与测试使用。
- 内存数据的强制覆写存在导致游戏闪退或存档逻辑错误的潜在风险。建议在使用任何功能前，妥善备份本地游戏存档。
